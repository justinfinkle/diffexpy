limma_py

Limma analysis in Python

Python is great for data analysis, but lacks some of the bioninformatics tools available in R (Bioconductor). limma_py is a simple package that uses rpy2 to run limma analyis. All input and output is with python and pandas, while the differential expresison analysis object takes care of the model fitting in R.
